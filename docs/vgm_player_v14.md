# VGM 播放器功能文档（v14 新增）

## 概述

v14 在原有 MIDI 播放器基础上，新增了完整的 VGM/VGZ/S98 芯片音乐播放器模块，
支持多芯片实时寄存器可视化。

---

## 新增文件

| 文件 | 说明 |
|------|------|
| `vgm_parser.cpp/h` | VGM/VGZ/S98 文件解析器 |
| `vgm_window.cpp/h` | VGM 播放器 UI 窗口（ImGui） |
| `vgm_chip_aliases.ini` | 芯片别名配置文件 |
| `vgm_folder_history.ini` | VGM 文件夹历史记录 |
| `vgm_player_state.ini` | 播放器状态持久化 |
| `docs/bugfix_sn76496_visualization.md` | SN76496 可视化 Bug 修复记录 |

---

## 编译方法

### 环境要求

- MinGW-w64 (g++ / gcc)
- CMake 3.15+
- Windows 10/11 x86_64

### 首次编译

```bash
bash build_v14.sh
```

脚本会自动：
1. 用 CMake 配置构建目录 `build/`（仅首次）
2. 增量编译所有变更文件
3. 输出 `ym2163_piano_gui_v14.exe`

### 强制全量重编

```bash
rm -rf build && bash build_v14.sh
```

### CMake 手动配置（可选）

```bash
cmake -B build -G "Unix Makefiles" \
    -DCMAKE_BUILD_TYPE=Release \
    -DCMAKE_CXX_COMPILER=g++ \
    -DCMAKE_C_COMPILER=gcc
cmake --build build --parallel $(nproc)
```

## 功能列表

### 1. 多格式文件支持

- **VGM**：VGM v1.00 ~ v1.71，支持所有主流命令
- **VGZ**：gzip 压缩的 VGM（自动解压）
- **S98**：S98 v1/v2/v3 格式

### 2. VGM 解析器（`vgm_parser.cpp`）

- 解析 VGM header（版本、数据偏移、循环信息）
- 读取 GD3 标签：曲名、游戏名、系统名、作者（中英文）
- 将寄存器写入命令提取为 `VgmEvent` 列表（tick / chip_type / addr / data）
- 支持 `CHIPTYPE_SN76496_SENTINEL (0xFF)` 区分 SN76496（DEVID=0x00）与无效条目

### 3. 文件浏览器

- 内嵌 ImGui 文件夹浏览，支持 .vgm / .vgz / .s98 过滤
- 文件夹历史导航（与 MIDI 播放器共享历史组件）
- 单击播放，双击进入目录

### 4. 电平表（Level Meter）

- 每个芯片通道独立显示音量条（底部向上填充）
- 按芯片类型分组，组间有视觉分隔
- **颜色随音量动态变暗**：亮度系数 = 0.4 + level × 0.6（level ∈ [0,1]）
  - 满音量：100% 颜色
  - 零音量以上：最低 40% 亮度，保持色相可辨识
- 静音通道显示暗灰色
- 通道标签显示在条内顶部

### 5. 钢琴键盘可视化

- 将芯片音符频率换算为 MIDI 音符编号，映射到钢琴键
- 按键高亮颜色来自对应芯片的通道色
- **亮度随音量调制**：亮度系数 = 0.5 + level × 0.5（最低 50%，防止颜色过淡）
- 多通道可同时映射到同一键（颜色叠加）

### 6. SN76496（PSG / Sega Master System）支持

#### 音调通道（CH0-CH2）

- 频率寄存器 → MIDI 音符 → 钢琴键映射
- 衰减寄存器（0=最大音量，0xF=静音）→ 电平条高度

#### 噪音通道（CH3）

| 控制位 | 说明 |
|--------|------|
| bits1:0 = 0/1/2 | 固定频率（低/中/高） |
| bits1:0 = 3 | 使用 CH2 的频率 |
| bit2 = 0 | 周期噪音（Periodic） |
| bit2 = 1 | 白噪音（White） |

- **标签**：`W0`/`W1`/`W2`/`W3`（白噪音）或 `P0`/`P1`/`P2`/`P3`（周期噪音）
- 使用 CH2 频率时（nfreq=3）：自动映射到对应钢琴键
- **颜色**：白噪音 = 灰色 `(160,160,160)`（无音高），周期噪音 = 青绿色 `(60,220,180)`（有音高）
- 固定频率 P0/P1/P2 也映射到钢琴键（分频 N/512、N/1024、N/2048）
- 固定频率 W0/W1/W2 同样映射钢琴键（以 LFSR 时钟频率作为光谱中心）

#### 初始化

上电复位时衰减寄存器全部初始化为 `0xF`（静音），符合 SN76496 硬件规格。

### 7. 芯片别名窗口

- 浮动窗口，可为每个芯片实例自定义显示名称
- 配置保存至 `vgm_chip_aliases.ini`

详见 [`docs/bugfix_sn76496_visualization.md`](bugfix_sn76496_visualization.md)

| # | 位置 | 问题描述 | 修复方式 |
|---|------|----------|----------|
| 1 | `vgm_parser.cpp` `ParseHeader()` | VGM v1.50 header 中 data offset 字段为 0 时未回退到规范默认值 0x40，导致解析 0 事件 | `if (!_header->dataOfs) _header->dataOfs = 0x40` |
| 2 | `vgm_parser.cpp` `VGM_CMD_INFO[]` | 行内注释截断 0x23-0x27 五个初始化条目，使整个命令表从 0x23 起全部错位 | 补全全部 8 个条目，注释移至行尾 |
| 3 | `vgm_window.cpp` `ResetShadowRegisters()` | SN76496 影子寄存器用 `memset(0)` 初始化，衰减值 0=最大音量导致噪音条始终显示满 | post-memset 循环将衰减寄存器设为 `0xF` |
| 4 | `vgm_window.cpp` | 三个 `if` 写在一行导致 `-Wmisleading-indentation` 编译警告 | 拆分为独立行，零警告编译 |

## 技术说明

### `VgmEvent` 结构

```cpp
struct VgmEvent {
    UINT32 tick;        // 事件时间（样本数）
    UINT8  cmd;         // VGM 命令字节
    UINT8  chip_type;   // 芯片设备 ID
    UINT8  chip_num;    // 0=第一片，1=第二片（双芯片 VGM）
    UINT8  port;        // 端口号
    UINT8  addr;        // 寄存器地址
    UINT8  data;        // 写入数据
    UINT16 data16;      // 16 位数据（部分命令）
};
```

### `VGM_CMD_INFO` 表

256 项固定大小数组，每项对应一个 VGM 命令字节（0x00-0xFF），记录：
- `cmdLen`：命令总字节数（含命令字节本身）
- `chipType`：目标芯片设备 ID（`CHIPTYPE_SN76496_SENTINEL=0xFF` 表示 SN76496）
- `size`：数据长度类型（0=固定，1=可变）

**注意**：此数组必须恰好包含 256 个初始化条目。行内 C++ 注释会截断后续条目，
需确保注释不影响初始化列表完整性。

### `ReadRelOfs()` 的二义性

返回值为 0 有两种含义：
- 存储的相对值本身为 0（VGM 规范：表示使用默认偏移）
- 绝对偏移确实为 0（通常无效）

调用方须在返回 0 时主动应用默认值，而非直接使用 0。

## 电平表颜色规范

### 颜色体系

所有通道颜色通过 `shiftHue()` 对同芯片内每个通道做微小色相偏移，使相邻通道可区分。
下表列出各芯片的**基础颜色**（channel 0 的颜色）。

### 通用颜色分类

| 颜色 | RGB | 用途 |
|------|-----|------|
| 绿色 | `(80,200,80)` | FM 合成通道（YM2612/2151/2413/2203/2608/2610/YMF262/3812/3526/Y8950/YMF271/YMF278B） |
| 蓝色 | `(100,180,255)` | PSG/SSG 音调通道（SN76496 T1-T3、YM2203/2608/2610 SSG、AY8910、C6280、POKEY除外） |
| 橙色 | `(255,160,60)` | ADPCM-A 采样通道（YM2608/2610） |
| 青绿 | `(60,220,180)` | SN76496 周期噪音（有音高） |
| 灰色 | `(160,160,160)` | 无音高噪音（SN76496 白噪音、SAA1099 noise-only 通道） |

### 芯片专属颜色

### 颜色一致性原则

- **同语义同色**：无音高噪音统一用灰色 `(160,160,160)`
- **电平条与钢琴键颜色来源相同**：均读取 `LevelMeterEntry.color`
- **亮度随音量变暗**：电平条 remap [0,1]→[0.4,1]，钢琴键 remap [0,1]→[0.5,1]
- **标签字母冲突**（如 `S`、`C`、`E`、`P`）在不同芯片不会同时出现于同一 VGM 文件，不构成实际冲突
